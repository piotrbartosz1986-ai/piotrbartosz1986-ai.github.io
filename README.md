# Root GitHub Pages (user site) — TWA Hello World

Ta paczka jest przygotowana pod repo o nazwie:

**piotrbartosz1986-ai.github.io**

Dzięki temu strona działa w root:
`https://piotrbartosz1986-ai.github.io/`

i Chrome zwykle pokazuje **„Zainstaluj aplikację”** (pełna PWA).

## Kroki
1. Utwórz repo: `piotrbartosz1986-ai.github.io` (public).
2. Wrzuć te pliki do root repo.
3. Settings → Pages:
   - Source: Deploy from a branch
   - Branch: main / (root)
4. Wejdź na stronę i zainstaluj.

## Test
- SW: kliknij „Sprawdź Service Workera”
- Offline: włącz tryb samolotowy i odpal z ikonki
